# 申请流程

申请流程的核心是信息管理。保研不是单线任务，而是多个院校、多个批次、多个材料版本和多个面试时间同时推进。

## 流程概览

```mermaid
flowchart LR
  A[搜集院校信息] --> B[准备材料版本]
  B --> C[联系导师或课题组]
  C --> D[提交夏令营/预推免申请]
  D --> E[参加笔试面试]
  E --> F[记录结果与复盘]
  F --> G[系统填报与确认]
```

## 建议表格字段

- 院校与学院。
- 专业方向。
- 导师或课题组。
- 批次类型。
- 截止时间。
- 材料清单。
- 提交状态。
- 面试时间。
- 结果与备注。
