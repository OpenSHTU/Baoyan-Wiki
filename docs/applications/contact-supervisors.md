# 联系导师

联系导师不是所有专业都必须，但如果目标方向导师制明显，提前沟通可以帮助你了解团队方向和招生情况。

## 什么时候联系

- 已经明确研究方向，并能说明自己的相关经历。
- 夏令营或预推免通知发布前后。
- 拿到入营、复试或优秀营员结果后。

## 邮件内容

建议包含：

- 简短自我介绍。
- 本科背景、排名和主要经历。
- 对导师方向的具体兴趣。
- 附件：简历、成绩单或代表性材料。
- 礼貌表达期待进一步交流。

## 注意事项

- 不群发明显模板邮件。
- 不夸大论文、项目和技能。
- 邮件标题清楚，例如“XX大学XX专业本科生申请推免咨询-姓名”。
- 没有回复很常见，可以间隔适当时间礼貌跟进一次。
