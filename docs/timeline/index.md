# 保研时间线

保研准备的难点不只是“做什么”，更是“什么时候做”。时间线页面用于把长期积累和申请季任务拆开，帮助你判断当前阶段最值得投入的事情。

## 总体节奏

- 大一到大二：稳定绩点与排名，探索专业兴趣，尽早形成可展示的课程、竞赛或项目积累。
- 大三上：明确方向，补齐英语、科研、竞赛、项目经历，开始关注目标院校往年要求。
- 大三下：准备夏令营材料，整理投递表格，按截止日期推进申请。
- 暑假：参加夏令营，复盘面试问题，更新材料和导师联系策略。
- 九月：参与预推免和九推，确认拟录取与系统填报。

## 关键原则

1. 官方通知优先。所有报名时间、材料要求、系统流程都以目标院校和教育部推免系统为准。
2. 提前维护材料。申请季最怕临时找证明、改格式、补章和改文件名。
3. 记录每次投递。建议维护表格，包含院校、专业、导师、截止时间、材料状态、结果和备注。
