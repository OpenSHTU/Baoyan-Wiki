# 大一到大三规划

保研竞争力来自长期积累，申请季只是把这些积累呈现出来。不同学校政策差异很大，但以下几类能力通常都值得尽早建设。

## 大一：先稳住基本盘

- 了解本专业保研政策：绩点计算、排名方式、加分项、处分与挂科限制。
- 建立学习习惯：优先保证核心课程成绩，避免后期为排名被动补救。
- 尝试加入课题组、社团技术组或竞赛队伍，重点是理解自己喜欢什么方向。

## 大二：形成可展示经历

- 选择一到两个方向深入：科研、竞赛、工程项目、社会实践或专业作品。
- 争取阶段性产出：论文投稿、竞赛奖项、项目报告、代码仓库、专利软著等。
- 规划英语考试：尽量不要把四六级、雅思、托福等任务拖到申请季。

## 大三：围绕申请补短板

- 整理个人经历：每段经历写清任务、方法、结果和个人贡献。
- 搜集目标院校：记录往年入营条件、面试形式、导师方向和材料要求。
- 提前打磨材料：简历至少准备通用版、科研版、工程版或专业方向版。

## 自查清单

- 是否清楚本学院保研资格和加分政策。
- 是否知道自己的排名区间和风险点。
- 是否有至少一段能讲清楚的科研、项目或竞赛经历。
- 是否有老师愿意为你写推荐信。
- 是否建立了目标院校信息表。
