# 推荐信与证明材料

推荐信和证明材料通常不难写，但最容易因为时间、盖章和格式耽误投递。建议提前准备并持续维护。

## 推荐信

联系推荐老师时可以提供：

- 个人简历。
- 成绩和排名信息。
- 申请方向与目标院校列表。
- 你希望老师重点评价的经历。
- 院校推荐信格式、提交方式和截止时间。

邮件或消息要简洁礼貌，留出足够时间，不要在截止前一天才请求老师帮忙。

## 证明材料

常见证明包括：

- 成绩单。
- 排名证明。
- 在读证明或学籍证明。
- 英语成绩证明。
- 奖项证书。
- 论文录用、发表或投稿证明。
- 项目、专利、软著、社会实践证明。

## 扫描与命名

- 扫描清晰，边缘完整，章印可辨认。
- 多页材料合并为一个 PDF。
- 文件名包含姓名和材料类型。
- 上传前打开检查，避免空白页、倒置页或错误版本。
