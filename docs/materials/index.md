# 材料准备

材料准备的目标不是堆满经历，而是让老师快速理解你是谁、适合什么方向、为什么值得进一步面试。

## 常见材料

- 简历：一到两页，突出教育背景、排名、科研、项目、竞赛和技能。
- 个人陈述：说明学术兴趣、经历脉络、未来计划和申请动机。
- 成绩单与排名证明：注意盖章、扫描清晰度和文件命名。
- 推荐信：提前沟通推荐老师，提供个人材料和申请方向。
- 其他证明：论文、奖项、英语成绩、项目作品、代码链接等。

## 文件管理建议

- 使用统一命名：`姓名-学校-材料类型-院校`。
- 建立投递文件夹：按院校和批次归档，避免误传旧版本。
- 保留可编辑源文件：简历、陈述、表格不要只保留 PDF。
- 做最终检查：页码、姓名、联系方式、日期、院校名称和专业名称。
