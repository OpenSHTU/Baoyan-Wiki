# 个人陈述

个人陈述用于展示你的学术兴趣、经历脉络和未来计划。它不需要写成宏大叙事，更重要的是真实、具体、连贯。

## 建议结构

1. 开头说明申请方向和核心动机。
2. 回顾与方向相关的课程、科研、项目或竞赛经历。
3. 解释你从经历中形成的问题意识和能力积累。
4. 说明未来研究兴趣、读研计划和与目标院校的匹配点。
5. 以简短段落收束，表达继续深造的意愿。

## 打磨方法

- 先列素材，再串逻辑，不要直接从空白页开始写。
- 每段经历都回答“我做了什么”和“我学到了什么”。
- 避免泛泛而谈，例如“贵校平台优秀”应替换为具体方向、团队或课程。
- 保持语气稳健，不要堆砌过度夸张的形容词。

## 检查清单

- 是否出现目标院校或专业名称错误。
- 是否与简历经历保持一致。
- 是否有明确方向，而不只是“热爱科研”。
- 是否控制在院校要求字数范围内。
