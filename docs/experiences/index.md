# 经验分享

经验分享的价值不只在“去了哪里”，更在“怎么判断、怎么准备、哪里踩坑”。本站鼓励记录有上下文的真实经历。

## 建议包含

- 申请年份、专业方向和本科背景概况。
- 成绩排名、英语、科研、竞赛、项目等基本情况。
- 投递院校和批次。
- 材料准备过程。
- 面试形式和题目。
- 最终选择与取舍。
- 遗憾、建议和想提醒后来者的事。

## 写作原则

- 可以匿名或模糊敏感信息。
- 不泄露未公开的面试材料和他人隐私。
- 尽量区分事实、感受和个人判断。
- 对失败经历同样友好，因为它们常常更有参考价值。
