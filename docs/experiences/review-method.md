# 复盘方法

复盘可以帮助你在申请季快速迭代，也能让经验分享更有参考价值。建议每次投递、面试或结果确认后尽快记录。

## 面试后立即记录

- 面试时间、时长和平台。
- 老师人数与面试流程。
- 自我介绍是否被打断。
- 专业课和英语问题。
- 简历中被追问的内容。
- 自己答得好的地方和卡住的地方。

## 阶段性复盘

每周可以检查：

- 哪些院校材料还没提交。
- 哪些材料版本需要统一更新。
- 哪些面试题重复出现。
- 哪些方向不再适合继续投入。
- 是否需要调整冲刺、匹配、稳妥名单。

## 结果复盘

无论成功还是失败，都建议记录原因假设。注意这是个人判断，不一定等于真实录取逻辑，但能帮助后来者理解申请过程中的不确定性。
