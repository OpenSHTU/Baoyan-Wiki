# 面试复习

面试复习不是背一套标准答案，而是把你的知识、经历和表达组织到能被追问的程度。

## 面试常见组成

- 自我介绍。
- 专业课问答。
- 英语问答或英文文献阅读。
- 科研、项目、竞赛经历追问。
- 综合素质与未来规划。

## 准备原则

- 简历上每一行都要能讲清楚。
- 专业课先抓核心概念、典型题和常见推导。
- 科研项目按“背景-问题-方法-结果-不足”准备。
- 英语回答不求复杂，优先保证清楚、准确、自然。
